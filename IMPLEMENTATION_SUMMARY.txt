================================================================================
                          RUNNING THE TESTS
================================================================================

RUN ALL TESTS:
  $ go test ./...

WITH RACE DETECTOR (RECOMMENDED):
  $ go test -race ./...

WITH COVERAGE:
  $ go test -cover ./...

SPECIFIC PACKAGE:
  $ go test -v ./pkg/vault/...

BENCHMARKS:
  $ go test -bench=. ./pkg/vault/...

ALL WITH VERBOSE OUTPUT:
  $ go test -v -race -cover ./...
